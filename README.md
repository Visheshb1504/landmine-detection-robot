# Landmine Detection Robot 

An intelligent robotic system designed to detect potential landmines using multiple sensors and machine learning.

## Overview

This project is inspired by real-world military applications and aims to enhance safety by detecting hazardous objects in terrain environments.

* Response Time: < 2 seconds
* Sensors Used: 10+ (Ultrasonic, Metal Detector, IR, Temperature, IMU)
* ML Integration: Edge Impulse (planned deployment on ESP32-CAM)
* Microcontroller: ESP32 And ESP32 CAM

---

## Features

* Real-time obstacle detection
* Metal detection for buried objects
* Multi-sensor fusion for accuracy
* Machine Learning-based classification (in progress)
* Autonomous navigation (planned)

---

## Tech Stack

* Embedded Systems: ESP32
* Programming: C/C++, Python
* Machine Learning: Edge Impulse
* Hardware: Sensors + Motor Driver + Li-ion Battery

---

## How It Works

1. Sensors continuously scan the environment
2. Data is processed by ESP32
3. ML model classifies potential threats
4. Alert system triggers (buzzer/LED)
